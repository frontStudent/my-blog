
### 导航栏添加新页签“代码片段”
1. 在docs文件夹下创建snippets文件夹
2. 在sidebars.js文件中添加：
```
snippetSidebar: [{type: 'autogenerated', dirName: 'snippets'}]
```
3. 在docusaurus.config.js - themeConfig - items 中添加：
```
{
    type: 'docSidebar',
    sidebarId: 'snippetSidebar',
    position: 'left',
    label: '代码片段',
}
```
也就是通过这种方式可以让docs中的一个文件夹独立出来变成导航栏上的一个页签
